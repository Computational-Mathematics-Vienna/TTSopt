This folder contains the Pyhton interfaces for solvers that have no Octave interface.

haltonset.py: can be used for solvers that use the Matlab function haltonset
runNomad.py: is the interface for Nomad
run_bobyqa.py, run_lincoa.py, run_uobyqa.py: are the interfaces for powell solvers
